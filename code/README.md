# Cloned Repositories

This directory contains key codebases for text steganography with LLMs.

## 1. Text-Steganography-Benchmark
- **URL**: https://github.com/redwoodresearch/Text-Steganography-Benchmark
- **Purpose**: A comprehensive benchmark for evaluating text steganography. Includes implementations of various methods and evaluation metrics.
- **Key Files**:
  - `steganography/`: Source code for steganography implementations.
  - `scripts/`: Benchmark running scripts.
- **Notes**: Developed by Redwood Research, it's a very solid baseline for evaluations.

## 2. LLM-Steganography
- **URL**: https://github.com/HighDoping/LLM-Steganography
- **Purpose**: Official implementation for the paper "Generative Text Steganography with Large Language Model" (2024).
- **Key Files**:
  - `llm_stega/`: Core logic for LLM-based steganography.
- **Notes**: Useful for modern black-box LLM steganography.

## 3. TrojanStego
- **URL**: https://github.com/worta/TrojanStego
- **Purpose**: Implementation related to the "TrojanStego" attack mentioned in recent literature (Meier et al., 2025).
- **Key Files**:
  - `trojan_stego/`: Implementation of the bit-bucket attack.
- **Notes**: Essential for studying persistent, fine-tuning-based steganographic attacks.
